# Boolean Function

Boolean function은 binary variable과 상수 0과 1, 그리고 logic equation symbol로 구성된 대수 표현식으로 나타낼 수 있다. 이 함수의 값은 항상 0 또는 1이다. 예를 들면 이런 식이다,

F1 = x + y'z

이런 걸 가지고 truth table 막 손으로 만들고 재밌는 gate 칭구들을 그렸던 기억이 난다. 정말 재.미.있.었.다. +는 뾰족한 OR 칭구로 묶고 둥그런 AND 칭구는 *, NOT은 세모 앞에 작은 점이 있는 칭구(Complement)로 바꿔서 열심히 그리면 된다.

# Canonical and Standard forms

binary variable은 normal form인 x와 complement form인 x'으로 나타낼 수 있다. 그러면 변수 x, y를 생각해보고 AND 연산자로 조합해보면 4가지 경우가 나오겠다. x'y', x'y, xy', xy 이런 칭구들은 minterm 또는 standard product라고 부른다. 반대로 OR 연산자로 조합한 애들은 maxterm 또는 standard sum이라고 한다.

중요한 특성으로 모든 Boolean function은 진리표 상에서 1을 뱉는 minterm들을 OR한 것으로 나타낼 수 있다. 예를 들어,

F1  = x'y'z + xy'z' + xyz
    = m1 + m4 + m7
F1' = x'y'z' + x'yz' + x'yz + xy'z + xyz'
    = m0 + m2 + m3 + m5 + m6

이렇게 sum of minterms로 나타낼 수 있다. (SOP, sum of products) 궁금하면 진리표를 직접 그려보는 수고를 해보자. F1'은 위와 같은데, F1'의 complement로 되돌려보자.

(F1')' = F1
       = (x + y + z)(x + y' + z)(x + y' + z')(x' + y + z')(x' + y' + z)
       = M0*M2*M3*M5*M6

기가 막히게도 product of maxterms로도 나타낼 수 있다. (POS, product of sum) 이렇게 나타낸 불형 함수를 canonical form을 갖는다고 한다.

F(x, y, z)  = Σ(1, 4, 7) = Π(0, 2, 3, 5, 6)

F'(x, y, z) = (m1 + m4 + m7)'
            = m1'*m4'*m7'
            = M1*M4*M7
            = Π(1, 4, 7)

그리고 이렇게 시그마랑 파이로 나타나타낼 수 있는데 그냥 뭐 딱 보면 알만한(?) 관계를 갖는다. 무엇이 minimal 하고 standard 한지 엄밀히는 잘 모르겠지만 별로 크게 알고싶지는 않으니 그만 알아보도록 하자.
