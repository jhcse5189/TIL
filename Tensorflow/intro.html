# Intro.

Computer Vision 수업 예제 중 원래 Colab에서 돌리는 건데 내 컴터에서 돌려보고 싶어서 설치해봤다. 환경은 Ubuntu 20.04 LTS에 python3만 설치되어있다고 가정한다.

$ sudo apt install python3-pip, python3-tk
$ pip3 install tensorflow
$ pip3 install matplotlib

$ python3 image_filtering.py

UserWarning: Matplotlib is currently using agg, which is a non-GUI backend, so cannot show the figure.

그냥 콘솔에서 돌리려고하면 GUI interface가 없기 때문에 tkinter도 설치해줘야한다.
